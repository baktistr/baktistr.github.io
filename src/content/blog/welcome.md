# Welcome

Welcome to my cybersecurity blog and documentation site! This is a space where I share my journey, learnings, and technical knowledge in the field of cybersecurity.

## What You'll Find Here

### Documentation
The **Docs** section contains structured, searchable technical content including:
- Pentesting methodologies and approaches
- Tool cheatsheets and reference guides
- CTF writeups and solutions
- OSCP notes and exam preparation materials
- Lab documentation and experiments

### Blog
The **Blog** features more personal and chronological content:
- Learning journey updates
- Career thoughts and experiences
- Random technical musings
- Quick tips and tricks
- Opinions on industry trends

## About This Site

This site is built with:
- **Astro 5** - A modern static site generator focused on performance
- **Tailwind CSS** - For styling with a dark-first design
- **MDX** - For rich, interactive content
- **Content Collections** - For type-safe content management

## Getting Started

Explore the navigation to browse through documentation and blog posts. Use the tags to find content related to specific topics.

Feel free to reach out if you find any issues or have suggestions!
