# Cheatsheets

Quick reference guides and cheatsheets for common penetration testing tasks and commands.

## Overview

Having quick access to common commands and options is essential during penetration testing. These cheatsheets provide the most commonly used commands for various scenarios.

### Categories

- **Enumeration** - Service and network enumeration commands
- **Exploitation** - Common exploitation techniques and payloads
- **Privilege Escalation** - Linux and Windows privesc techniques
- **Active Directory** - AD enumeration and attack techniques
- **Web Application** - Web testing techniques and payloads
