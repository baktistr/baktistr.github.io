# Exploring Markdown Styles in Obsidian

Welcome to this exploration of all basic Markdown styles. Markdown allows you to format your notes easily in Obsidian. Let’s go through the common elements.

## 1. **Headings**

Headings are used to create sections and subsections in your notes.

### Heading 3
#### Heading 4
##### Heading 5

## 2. **Bold and Italics**

You can emphasize text using **bold** or *italics*:

- **Bold**: `**This text is bold**`
- *Italics*: `*This text is italic*`
- Combined: `***Bold and italic combined***`

## 3. **Lists**

### Unordered List

- Item 1
- Item 2
  - Sub-item 1
  - Sub-item 2

### Ordered List

1. First item
2. Second item
3. Third item

## 4. **Links**

You can include links to websites or internal notes.

- [External Link to Google](https://www.google.com)
- [Link to a note in Obsidian](obsidian://open?vault=MyVault&file=MyNote)

## 5. **Images**

Images can be embedded with the following syntax:

![Alt Text](https://example.com/image.jpg)

## 6. **Code**

You can include both inline and block code.

### Inline Code
Use backticks for inline code: `` `inline code` ``

### Block Code

```python
def hello_world():
    print("Hello, world!")
```

## 7. **Blockquotes**

Blockquotes are used to highlight important information.

> This is a blockquote. You can use it for quotes or important notes.

## 8. **Callouts**

Callouts help emphasize specific information. Here are some examples:

### Note

> [!note] Important Note
> Remember to backup your work regularly.

### Warning

> [!warning] Caution
> Make sure to check for updates before proceeding.

### Tip

> [!tip] Helpful Tip
> When using Obsidian, you can link your notes using `[[note name]]`.

### Success

> [!success] Success!
> You've completed the Markdown tutorial.

## 9. **Tables**

You can organize data in tables for easy reading.

| Name   | Age | Location  |
|--------|-----|-----------|
| Alice  | 30  | London    |
| Bob    | 25  | New York  |
| Charlie| 35  | Tokyo     |

## 10. **Strikethrough**

You can use strikethrough to indicate deleted or incorrect text.

- `~~This is a strikethrough~~` will show as ~~This is a strikethrough~~

## 11. **Task Lists**

You can create a task list using checkboxes.

- [ ] Task 1
- [x] Task 2 (completed)
- [ ] Task 3

## Conclusion

This is just a basic overview of the different Markdown styles in Obsidian. You can combine these styles to create rich, well-structured notes!
